# Day07-d3CSS选择器-PxCook软件-盒子模型

## 1. 结构伪类选择器
1. 结构伪类选择器作用
   - 根据元素的结构关系查找元素
   ```html
   选择器             说明
   E:first-child      查找第一个E元素
   E:last-child       查找最后E元素
   E:nth-child(N)     查找第N个E元素(第一个N元素值为1)
   ```
2. E:根据元素的结构关系查找多个于元素
```html
功能                公式
偶数标签             2n
奇数标签             2n+1， 2n-1
找到5的倍数的标签     5n
找到第5个以后的标签   n+5
找到第5个以前的标签   n-5
```

3. 伪元素选择器
   - 作用创建虚拟元素(伪标签)，用来摆放装饰性的内容
   ```html 
   选择器     说明
   E::before  在E元素里面最前面添加一共伪元素
   E::after   在E元素里面最后面添加一共伪元素
   ```

   - 注意点
          - 必须设置content:""属性，用来设置伪元素的内容，如果没有内容，则引号留空即可
          - 伪元素默认为是行内显示模式
          - 权重和标签选择器相同

## 2. PxCook
4. PxCook(像素大厨)是一款切图设计工具软件，支持PSD文件文字，颜色，距离自动识别
   - 开发面版(自动智能识别)
   - 设计面板(手动测量尺寸和颜色)

## 3. 盒子模型
5. 盒子重要组成部分
   - 内容区域 - with & hight
   - 内边距   - padding(出现在内容与盒子边缘之间)
   - 边框线   - border
   - 外边框线 - margin(出现在盒子外面)

6. 边框线
   - 属性名：border (bd)
   - 属性值：边框线的粗细 线条样式 颜色 (不区分顺序)
   - 常用的属性值
   ```html
   属性值   线条样式
   solid   实线
   dashed  虚线
   dotted  点线
   ```

7. 设置当方向的边框线
   - 属性名：border-方位名词 (bd + 方位名词的首字母，例如 bdl )
   -属性值： 边框线粗细 线条样式 颜色 (不区分顺序)

8. 盒子模型-内边距
   - 作用:设置 内容 与 盒子边缘 之间的距离
   -属性名:padding /padding-方位名词

9. 盒子模型- 内边距-多值写法
   ```html
   取值个数     示例                                含义
   一个值       padding:10                         四个方向内边距均位10px
   四个值       padding:10px 20px 30px 40px        上:10px 右20px 下:30px 左:40px
   三个值       padding:10px 40px 80px             上:10px 左右:40px  下:80px
   两个值       paddingL10px 80px                  上下:10px    左右:80px
   ```
   
   - 记忆方法 从上面开始逆时针转一圈,如果当前没数值 取值跟对面一样

10. 盒子模型-尺寸计算
    - 默认情况
    盒子尺寸=内容尺寸+border尺寸+内边距尺寸

    - 结论
    给盒子加border/padding会撑大盒子

    - 解决方法
      - 手动做减法,减掉broder/padding尺寸
      - 内减模式,box-sizing:border-box
