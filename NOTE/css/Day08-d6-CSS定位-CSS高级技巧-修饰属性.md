# Day08-d6-CSS定位-CSS高级技巧-修饰属性

## 1. 定位
1. 定位
   - 作用：灵活改变盒子在网页中的位置
   - 实现
     - 定位模式：position
     - 扁偏移：设置盒子的位置
      - left
      - right
      - top
      - botton

2. 相对定位
   - position：relative

3. 绝对定位
   - position:absolute
   - 使用场景：子级绝对定位，父级相对定位(子绝父相)

4. 定位居中
   - 实现步骤 
     - 绝对定位
     - 水平，垂直边扁移50%
     - 子级向左 上移动自身尺寸的一半
     - 左上的外边距为-尺寸的一半
     - transform：translate(50%,50%)

5. 固定定位
   - position:fixed
   - 场景：元素位置在网页滚动时不会改变

6. 定位总结

   | 定位模式 |   属性值   | 是否脱标     |  显示模式    |  参照物 |
   | -------- | ---- | ---- | ---- | ---- |
   | 相对定位 | relative | 否 | 保持标签原有显示模式 | 自己原来位置 |
   | 绝对定位 | absolute | 是 | 行内块特点 | 1.已经定位的祖先元素 2.浏览器可视区 |
   | 固定定位 | fixed | 是 | 行内块特点 | 浏览器窗口 |

## 2.CSS高级技巧
7. CSS精灵
   - CSS精灵，也叫CSS Sprites,是一种网页图片应用处理方式。把网页张一些背景图片整合到一张图片文件中，
     在用background-position精确的定位出背景图片的位置
   - 优点减少服务器被请求次数，减轻服务器的压力，提高页面加载速度

8. 实现步骤
   - 创建盒子，盒子尺寸与小图尺寸相同
   - 设置盒子背景图为精灵图
   - 添加background-position属性，改变背景图的位置
     - 使用PxCook测量小图左上角坐标
     - 取负数坐标为background-position属性值(向左上移动图片位置)

9. 字体图标
   - 展示的是图标，本质是字体
   - 作用：在网页中添加简单的，颜色单一的小图标
   - 优点
     - 灵活性：灵活的修改样式，例如尺寸，颜色
     - 轻量级：体积小，渲染快，降级服务器请求次数
     - 兼容性：几乎兼容所有主流浏览器
     - 使用方面：先下载在使用

10. 字体图标-使用字体
    - 引入字体样式表
    - 标签使用字体图标类名
      - iconfont:字体图标基本样式(字体名，字体大小等)
      - icon-XXX：图标对应的类名

11. 字体图标-上传矢量图
    - 作用：项目特有的图标上传到iconfont图标库，生成字体



## 3. 修饰属性
12. 垂直属性 vertical-align
   属性名 
    ```
   属性值       效果
   baseline     基线对齐(默认)
   top          顶部对齐
   middle        居中对齐
   bottom        顶部对齐
    ```

13. 过渡transition
    - 作用：可以伪一个元素在不同状态之间切换时候添加过度效果
    - 属性名：transition(复合属性)
    - 属性值：过渡属性 花费的时间(s)
    - 提示
         - 过渡的属性可以是具体的CSS属性
         - 也可以为all(两个状态的属性值不同的所有属性，都产生过度效果)
         - transition设置本身

14. 透明度属性(opacity)
    - 作用：设置整个元素的透明度(包括背景和内容)
    - 属性名opacity
      ```
      属性值   
      0~1
      0：完全透明(元素不可见)
      1：不透明
      0~1之间的小数：半透明
      ```

15. 光标类型-cursor
    - 作用：鼠标悬停在元素上时指针显示的样式
    - 属性名：curdor
    ```
   属性值     效果
   default   默认值 通常是箭头
   pointer   小手效果，提示用户可以点击、
   text      工字型，提示用户可以选择文字     
   move      十字光标，提示用户可以移动
    ```
